# Install dependencies
1. Install dependencies modules
```
$ brew install node
$ brew install watchman
```
2. Xcode & CocoaPods
    - Install iOS Simulator and XCode
    - Install ```CocoaPods``` 
        - ```$ sudo gem install cocoapods```

# Start
 1. Open Terminal ```$ yarn run start```
 2. Open New Termianl ```$ yarn run ios```
